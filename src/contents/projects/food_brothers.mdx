Food Brothers is a complete restaurant ordering solution designed for an upscale burger establishment in Amsterdam. The website features a premium aesthetic with smooth animations, intuitive navigation, and a seamless ordering flow that takes customers from menu browsing to checkout in just a few taps.

The application is built with a modern tech stack optimized for performance and developer experience:

- **Next.js 16** with App Router for server-side rendering and routing
- **React 19** for the UI layer
- **Tailwind CSS 4** for styling
- **Radix UI** for accessible component primitives
- **Zustand** for lightweight state management (cart, checkout, language)
- **React Hook Form + Zod** for form handling and validation
- **Vercel** for deployment and analytics

> ## Key Features

### Shopping Cart System

The cart system is designed for seamless user experience:

- **Persistent Cart** - Items saved across page navigation using Zustand
- **Floating Cart Button** - Always-visible cart indicator with item count and total
- **Slide-out Drawer** - Full cart management without leaving the current page
- **Topping Details** - Each item shows its customizations clearly

> ## Lessons Learned

- **State Management Simplicity** - Zustand proved to be the perfect choice for e-commerce state. Its simplicity over Redux made cart logic straightforward to implement and debug.

- **Mobile-First Pays Off** - Starting with mobile design and scaling up resulted in a much cleaner codebase than retrofitting responsive design later.

- **Radix UI for Accessibility** - Using Radix primitives for modals, dropdowns, and sheets meant accessibility came almost for free, significantly reducing development time on a11y features.

- **Form Validation UX** - Combining React Hook Form with Zod schemas creates an excellent developer and user experience. Real-time validation feedback keeps users informed without being intrusive.
